# Project Evaluations

This directory contains the code for the public dashboard and the data showed in it. The dashboard was built using Vite and React, and the data is stored in JSON files. The production dashboard can be found at [https://private-transfers.pse.dev/]()

# Setup

```bash

# 1. Clone the repository
git clone https://github.com/privacy-ethereum/private-transfers-benchmarks

# 2. install dependencies
cd private-transfers-benchmarks
pnpm install

# 3. Navigate to the specific directory
cd project-evaluations

# 4. Start the web server to view the dashboard and data
pnpm run dev
```
